# 🐍 Snake Game

A modern take on the classic **Snake Game**, built using **React.js** and styled with **Tailwind CSS**. This responsive and interactive game brings a nostalgic experience to the browser — play anytime, anywhere!

## 🚀 Live Demo

🎮 **Play now on Netlify:** [Click Here to Play]  (https://veerus-snakegame.netlify.app/)
---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Deployment:** Netlify

---

## 🎮 Features

- ✅ Classic snake movement and growth
- ✅ Keyboard controls (Arrow Keys)
- ✅ Game-over detection
- ✅ Score tracking
- ✅ Responsive layout for desktop & mobile
- ✅ Clean and modern UI with Tailwind CSS

## 🧑‍💻 Local Setup

To run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/your-username/snake-game.git
cd snake-game

# 2. Install dependencies
npm install

# 3. Start the development server
npm start


🗂️ Folder Structure

src/
├── components/       # Game components
├── assets/           # Images, icons, etc.
├── App.jsx           # Main app
├── index.css         # Tailwind CSS
└── main.jsx          # Entry point


📦 Deployment
Deployed using Netlify. Just drag and drop the production build/ folder, or connect your GitHub repository for continuous deployment.
